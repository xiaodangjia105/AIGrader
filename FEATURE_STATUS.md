## 二、有问题/待完善的功能 ⚠️

### 2.1 AI 相关

| 问题 | 影响 | 说明 |
|---|---|---|
| **DeepSeek API Key 未配置** | 🔴 主观题批改不工作 | 当前使用占位符 `sk-placeholder`，需要真实 API Key。客观题（选择/判断）不受影响 |
| **填空题 LLM 调用未充分测试** | 🟡 可能不稳定 | DeepSeek 返回格式异常时降级为默认评分 |
| **主观题 Prompt 未调优** | 🟡 评分质量一般 | Prompt 模板为基础版本，未经过实际教学场景调优 |
| **向量检索未实现** | 🟡 pgvector 扩展已安装但未使用 | 表中有 embedding 字段但未填充实际向量，相似题检索功能待开发 |
| **Redis 批改缓存未验证** | 🟡 代码已写但未充分测试 | `getCachedResult` 方法目前返回 null |

### 2.2 前端相关

| 问题 | 影响 | 说明 |
|---|---|---|
| **教师复核页面路由问题** | 🟡 点击"查看提交"传的是 assignmentId 而非 submissionId | TeacherAssignments 跳转 `/teacher/grading/{assignmentId}`，但 GradingReview 按 submissionId 渲染——实际加载的是该作业的所有提交列表，功能可用但命名不一致 |
| **前端大 chunk 1.2MB** | 🟡 首屏加载偏慢 | Ant Design 全量引入，后续需按需加载 |
| **移动端未适配** | 🟡 MVP 阶段不做 | 仅在桌面端测试 |

### 2.3 后端相关

| 问题 | 影响 | 说明 |
|---|---|---|
| **无用户认证** | 🟡 MVP 故意不做 | 使用 Demo 账户选择登录，任何人都可访问所有接口 |
| **无 API 限流** | 🟡 高风险 | 未配置 Rate Limiting，可能被滥用 |
| **无全局异常处理** | 🟡 体验一般 | 异常直接返回 500 + 堆栈，未做友好封装 |
| **CORS 过于宽松** | 🟡 安全问题 | `allowedOriginPatterns("*")` 允许所有来源 |
| **spring.jpa.open-in-view 警告** | 🟢 性能警告 | 默认开启，可能在视图渲染时保持数据库连接 |
| **application.yml 含明文密码** | 🟡 安全问题 | 默认值 `ai_grader123` 可见，生产环境需外部化 |
